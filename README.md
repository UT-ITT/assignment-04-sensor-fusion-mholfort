# assignment-04-CV-Sensor-Fusion

## image_extractor.py
- Start image extractor via command line with image path, output path, width and height parameters
    - Example: python image_extractor.py sample_image.jpg warped_image.jpg 600 800
- Place points in the following order:
    1. Top left corner
    2. Top right corner
    3. Bottom right corner
    4. Bottom left corner
- Press ESC to reset/exit
- Press S to save new image

## AR_game.py
- Run the AR game code
- Hold your board with aruco markers to the camera
- Once the aruco markers are detected the game will start
- Pop the circles from below using your finger
- Press Q to exit game
